
Cookie Consent
========

Implementierung des Cookie Hinweises für die EU-Richtlinie 2009/136/EG (E-Privacy-Richtlinie).


Paketname:

    quiqqer/cookiesend


Features (Funktionen)
--------

- Vorgefertigter Text kann ersetzt werden
- Cookie Hinweisanzeige kann angepasst werden


Installation
------------

Der Paketname ist: quiqqer/*
